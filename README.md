# react-native-horizon-sdk
Horizon SDK wrapper for React Native
## Installation

```sh
npm install react-native-horizon-sdk```

## Usage

```js
import { HorizonSdkView } from "react-native-horizon-sdk";

// ...

<HorizonSdkView />
```
